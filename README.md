## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:
  * A basic understanding of Ruby and Git
### Usage
1. Install Ruby
2. Install Rails
3. Clone this repo 
```
git clone https://github.com/mershark/hello-rails-react.git
cd hello-rails-react
```
```
run
rails server
```
